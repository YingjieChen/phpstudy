## 如何通过linux命令行发送email
* 首先要在linux 机器上面安装mailutils,然后安装shareutils组件
* 再在linux 命令行下面执行 mail -s "邮件主题" XXXX@域名 
##如何用PHP写出弧形文字
* 首先安装		PHP5 apt-get install php5
* 开启GD库		apt-get install php5-gd
* 安装apache2服务器	apt-get install apache2
* 详细的操作方法请查看代码库
